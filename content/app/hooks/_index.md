This section provides comprehensive documentation for all custom React hooks provided by the `@fireact.dev/app` package. These hooks encapsulate reusable logic and stateful behavior, simplifying common tasks in your Fireact frontend application.

### Key Hooks:

*   **[useAuth](/app/hooks/useAuth/)**: Provides convenient access to the `AuthContext`.
*   **[useConfig](/app/hooks/useConfig/)**: Provides convenient access to the `ConfigContext`.
*   **[useLoading](/app/hooks/useLoading/)**: Provides convenient access to the `LoadingContext`.
*   **[useSubscription](/app/hooks/useSubscription/)**: Provides convenient access to the `SubscriptionContext`.
*   **[useSubscriptionInvoices](/app/hooks/useSubscriptionInvoices/)**: Provides a convenient way to fetch and paginate a list of invoices associated with a specific subscription.
